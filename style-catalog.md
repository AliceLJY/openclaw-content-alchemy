## 组合规则

风格可以两两组合，产生独特效果。组合格式：`{风格A的prompt后缀}, combined with {风格B的prompt后缀}`

**推荐组合（已验证好看）**：

| 组合 | 效果 | 适合 |
|------|------|------|
| 赛博朋克 + 浮世绘 | 赛博和风，科幻与传统碰撞 | AI + 东方文化 |
| 梵高 + 赛博朋克 | 旋涡星空遇见霓虹都市 | 科技与人文冲突 |
| 黏土定格 + 赛博朋克 | 反差萌，可爱小人在赛博世界 | 轻松科技话题 |
| 毛毡手工 + 水墨 | 温暖材质+东方意境 | 治愈系中国风 |
| 波提切利 + 草间弥生 | 古典优雅遇当代波点 | 古今对话 |
| 敦煌壁画 + 赛博朋克 | 千年佛窟遇霓虹未来 | 传统与科技 |
| 霍珀 + 蒙克 | 都市孤独 × 存在焦虑，双重暴击 | 现代人精神困境 |
| 马蒂斯 + 丰子恺 | 东西方的童真与简笔交融 | 生活哲学 |
| 沃霍尔 + 年画版画 | 波普消费遇民俗热闹 | 中国消费文化 |
| 班克斯 + 巴斯奎特 | 街头涂鸦双暴击 | 社会批判 |

**组合原则**：
- 最多组合2种（3种以上会糊）
- 选反差大的组合更有视觉冲击力
- 同一色调的风格组合容易平淡，跨色调才出彩
